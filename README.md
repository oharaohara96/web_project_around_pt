🏙️ Around The U.S.
Um projeto interativo de rede social onde os usuários podem compartilhar fotos de lugares incríveis, curtir postagens e editar suas informações de perfil. Este projeto faz parte do currículo de desenvolvimento web da TripleTen e foca na criação de interfaces dinâmicas e funcionais.

🚀 Funcionalidades
Edição de Perfil: Alteração dinâmica de nome e profissão através de janelas pop-up (modais).

Adição de Cartões: Possibilidade de adicionar novos locais informando título e URL da imagem.

Curtir Postagens: Sistema de "like" interativo nos cartões de fotos.

Exclusão de Cartões: Botão para remover postagens da lista.

Visualização de Imagens: Ao clicar em uma foto, um pop-up de visualização em tela cheia é aberto com a legenda correspondente.

Validação em Tempo Real: Formulários com validação nativa de JavaScript que bloqueiam o envio se os campos estiverem incorretos ou vazios.

Fechamento Inteligente: Pop-ups podem ser fechados clicando no "X", clicando no overlay (fora do conteúdo) ou pressionando a tecla Esc.

🛠️ Tecnologias e Conceitos Utilizados
HTML5 & BEM: Estruturação utilizando a metodologia Block Element Modifier para um código mais organizado e escalável.

CSS3 Avançado: Uso de Grid Layout, Flexbox, Media Queries (responsividade) e posicionamento absoluto/relativo.

Manipulação de DOM: Criação de elementos dinâmicos usando a tag <template>, clonagem de nós e renderização de listas a partir de arrays de objetos.

JavaScript (ES6): * Uso de addEventListener para múltiplos tipos de interação.

Lógica de validação personalizada.

Refatoração de funções para maior reaproveitamento de código.

📱 Responsividade
O design foi construído para ser totalmente responsivo, garantindo uma excelente experiência de usuário desde telas de desktop de 1280px até dispositivos móveis de 320px.
